# Generator certyfikatów - Excel VBA

Aplikacja zaimplementowana obiektowo w Excel VBA. Generator certyfikatów wraz z systemem zapisowym na szkolenia.

### Opis:

<p align="justify">
Aplikacja charakteryzuje się ograniczonymi do minimum bezpośrednimi operacjami na danych arkuszy. Arkusze służą jako kontenery danych. Po rozpoczęciu sesji dane odczytywane są do kolekcji, przed zamknięciem zapisywane do arkuszy. Aplikacja została napisana w pełni obiektowo, obsługuje tworzenie, modyfikację, usuwanie uczestników oraz szkoleń, system zapisowy na zajęcia, generowanie certyfikatów oraz drukowanie ich poprzez przekazanie danych wybranych pozycji do szablonów dokumentu Word.
</p>

### Wykorzystane technologie:

- VBA <br />

### Wykorzystane narzędzia:

- Microsoft Excel 2016 <br />
